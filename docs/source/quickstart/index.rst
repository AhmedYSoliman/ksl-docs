=========================
Quickstart
=========================

The instructions here are for all the users how have obtained credential (username/password) to access KSL resources.
The pages in this section provide simple steps how to get onto the KSL systems and submit workloads.

If you are familiar with HPC clusters and need a quick reference on the specifics of how to interact with KSL computational resources, you will find the relevant information here in a concise form. 
For details, please explore the other sections of the documentation starting from :ref:`available_systems`.
